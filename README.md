# Eletronic urna frontend system

* An eletronic urna application built during the course of [b7web](https://www.b7web.com.br)

Subjects Learned:

* DOM Manipulation;
* Classes Manipulation
* EventListeners
* Display messagens without blocking the entire browser ( like alert() );
